MASTER CANVA THUMBNAIL TEMPLATE — starter kit
================================================

15 blank starter pages (numbered in your requested order), each at Etsy's
recommended 2700x2025 photo size, with a dashed placeholder frame and a
faint page-name label so you always know which is which while building in
Canva. Delete the label/frame once you start designing each page — they're
just guides.

elements/ — reusable pieces to drop onto any page, all transparent PNGs
except the background:
  - device-mockup-laptop.png   A laptop frame with an empty white screen.
                                 Drop your app/PDF screenshot into the white
                                 area, or use Canva's "replace image" on a
                                 duplicate of it as a placeholder frame.
  - badge-circle-stat.png       A circled stat callout ("00 / Replace Me") —
                                 duplicate it per page and swap in a real
                                 number + label (e.g. "36 / Peptides Tracked").
  - checkmark-row.png           The italic checkmark trust row
                                 ("✓ Instant download · ✓ Printable + app · ...").
  - background-texture-warm.jpg The warm gradient + grain background alone,
                                 for any page that needs it as a base layer.
  - decorative-glow-accent.png   A soft radial glow accent — layer behind a
                                 mockup or badge for depth.

What's NOT included, and why:
  - Icons: Canva's own built-in icon library has thousands of options —
    there was no need to manufacture a custom icon set for a starter
    template. Search Canva's Elements tab for anything feature-specific
    (syringe, heart, moon, etc.).
  - Lifestyle photography / photo overlays: this shop does not have
    licensed stock photos (hands, coffee cups, cozy rooms, etc.), and we
    won't fabricate AI "people" photos to fake them. If you want the
    Lifestyle page (12) to use real lifestyle photography, license a few
    photos (Etsy's own marketplace, a stock site, or your own photos) and
    drop them in — everything else in this kit still works without them.

Colors used throughout (for consistency if you add more elements):
  Headline / callout text: #7A1524 (deep wine)
  Eyebrow / subhead text:  #A8481F (warm terracotta)
  Accent 1 (coral badge):  #E4572E / #B83A22
  Accent 2 (gold badge):   #D9A22C / #A6721B
  Background gradient:     #FDF3E0 -> #F8DFA9 -> #F0C97D
  Laptop bezel:            #3A3430 -> #221D1A
Fonts: "Fraunces" (serif, headlines/eyebrows) + "Figtree" (sans, badge
numbers/labels) — both free on Google Fonts and available in Canva's font
picker by searching those exact names.
